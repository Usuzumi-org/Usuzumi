# Changelog

All notable changes to Mashiro UI will be documented in this file.

This project follows a simple `major.minor.patch` version pattern:

- Major: breaking class names, markup contracts, token removals, or behavior changes.
- Minor: new components, page patterns, tokens, or documented states.
- Patch: bug fixes, visual refinements, documentation corrections, and validation updates.

## Unreleased

### Added

- Initial zero-build UI library entry files.
- Split CSS source files for tokens, base styles, typography, components, layout, patterns, and utilities.
- Design system catalog, personal homepage example, and app introduction example.
- Framework-free JavaScript for theme toggles, language toggles, and custom select behavior.
- Repository validation script for syntax, local path references, and design-system guardrails.

### Changed

- Documentation now separates project usage, design rules, contribution guidance, and third-party notices.
